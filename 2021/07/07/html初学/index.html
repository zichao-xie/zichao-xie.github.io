html初学

2021-07-07- 2021-07-12
977- 4m
学习笔记-html

简介

  • 超文本标记语言。
    • 超文本：一种组织信息的方法，通过超级链接将多种媒介关联起来。
    • 标记：标签。用<>包裹。
  • 静态与动态网页
    • 静：页面内容和显示效果不会改变
    • 动：不改变代码，显示内容随时间，环境，数据库操作结果而改变。
  • 浏览器
    • 解释和执行HTML源码的工具
    • 五大主流
      • IE
      • Firefox-火狐
      • Chrome-谷歌
      • Opera-欧朋
      • Safari-苹果
    • 内核
      • Trident：IE
      • Gecko：firefox
      • webkit：苹果研发的，Safari，chrome
      • Presto：Opera，13年之后加入谷歌，弃用了
      • Blink：chrome，Opera研发的新的内核，13年之后使用

结构

块级元素

  • 单独一行。
  • 高宽可控，默认与父元素等宽，即铺满整行。
  • 内可包含块级元素与行级元素。
  • 默认display:block，通过将其变为display:inline可以变成行级标签。

行级元素

  • 不会独占一行，与相邻行级元素占同一行，直到行满，到下一行。
  • 宽高不可被设置。
  • 内只能包含行级元素。
  • display:inline

W3C-world wide Web

  • 块可以包含块和内联，行只可以内联。
  • 块不能放到<p>里
  • 只能放内联
  • 块与块并列，行与行并列
  • 属性用双引号
  • JavaScript加上language和type属性
  • 元素用小写
  • 使用实体转义
  • 给所有的属性赋值，如不，值是属性名本身
  • 标签必须闭合
  • form要有label

基本样式

  • <div>：用来布局，没有含义，单独一行。
  • <h1>：标题，1-6级。
  • <p>：段落，段落中间的回车会有一个空格，不会显示换行。
  • <br>：换行，单标签，可以使用<br/>闭合，没有属性。
  • <hr width="80%">：产生一个分割线，单标签。
  • <a href="">：设置超文本链接。
    • target属性
      • _blank：新窗口
      • _self：当前窗口
      • _parent：？
      • _top：？
  • <img>：用来加载图像
    • src：图片路径
    • alt：当图片加载不成功时，显示其内容。
    • title：鼠标放上去后，提示文字。
  • <span>：与<div>一样用来布局，但不会单独占一行。
  • <ul>：无序列表
  • <ol>：有序列表
  • <li>：列表内容
  • <!---->：注释
  • <table>

文本格式化标签

  • <b>：加粗，行级
  • ``：加粗，行级，强调，看起来和`是一样的。用于seo时，便于提取关键字。
  • <i>、<em>：倾斜，<em>强调
  • <pre>：块级，预格式化文本，保留换行 空格 宽度，文字字号会小一号。
  • <small>：行级，浏览器支持的最小字号为12px，要显示更小的需要做处理。
  • <big>：行级，被淘汰了，但可以使用。
  • <sub>：下标
  • <sup>：上标
  • <q>：短文本引用
  • <address>：网页加入地址信息，字体会倾斜。
  • <caption>：标题

属性

  • 通用属性
    • title
    • id
    • class
    • style
  • 自定属性
    • 格式data-*
  • 自有属性

实体转义

  • &nbsp;：空格
  • &lt;：<
  • &gt;：>
  • &amp;：&

文件目录命名规范

  • 不能出现汉字，通常字母或下划线开头

webstorm快捷键

  • link:css+tab导入外部css文件

  • !+Tab生成HTML文件基本结构或html:5+Tab

  • ul>li{li$}*3 ：生成表格。

  • Ctrl+d：复制并粘贴。

  • Ctrl+/：注释

  • table>tr*2>td{内容$}*3：生成表格
