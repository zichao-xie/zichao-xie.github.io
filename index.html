快速掌握新语言

Posted on 2020-12-14

  • 接触新语言先定性，是否是图灵完备语言（标记语言不是）。

    在可计算性理论里，如果一系列操作数据的规则（如指令集、编程语言、细胞自动机）按照一定的顺序可以计算出结果，被称为图灵完备（turing complete）。

  • 语言历史，语言擅长做什么，为什么被发明出来？

    • 看文章、博客
    • github看开源项目

  • 程序逻辑

    • 变量命名（语言规范、程序猿规范）
    • 循环
    • 分支

  • 变量

    • 静态、动态（声明变量时不用指定类型是动态类型）
    • 强类型、弱类型（根据值，可以知道是什么类型。例如，1是整形，1.0是浮点数，Python是强类型。如果只传地址，无法判断变量类型）
    • 值类型、引用类型（影响if判断时，a==b，值类型判断数值，引用类型判断指针是否指向同一个地址）
    • 字符串(Java 两个String值相等的话，==为1，都在常量池里面，stringbuffer不是)
      • 正则表达式
      • 时间、日期（支持不支持计算？格式、时区、时间戳？）

  • 容器（进阶）

    • 基本操作
    • 浅复制、深复制
    • 序列化、反序列化

  • 抽象方法的定义、实现

    • 函数
    • 类（继承、封装、多态、接口）

  • 生态（进阶）

    • 第三方包管理工具（maven）

  • 高级功能

    • Java 高性能，高线程，携程？

hexo解决图片显示问题

Posted on 2020-12-14

安装好hexo后，发现无法插入图片，解决办法如下

  • 安装图片插件

    报错如下：

    ​ 显示权限不够，但是我已经加了sudo。查阅博客后发现，是因为用root 用户进行了局部安装npm包的操作，留下所属权为 root 的文件，导致普通用户无法访问 root的文件内容。找到报错的文件，会看到它的所有者是 root。之前用 root 进行了局部的安装操作，导致这个文件的所有者是 root ，还包括 .npm 文件夹下部分文件夹的所有权，也是 root，普通用户当然就无权访问了。就会报权限错误。

    解决：

    ​ 将.npm所有者变为自己

  • 设置typora，使用相对路径。

  • 修改_config.
